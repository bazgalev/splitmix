# SplitMix

Браузерный инструмент для нарезки аудиофайлов на треки — без загрузки на сервер, всё прямо в браузере.

**[Открыть демо →](https://bazgalev.github.io/splitmix/)**

## Что делает

1. Загружаешь аудиофайл (MP3, WAV, OGG, FLAC, AAC)
2. Расставляешь маркеры на форме волны — они показывают, где резать
3. Выбираешь формат экспорта и нажимаешь «Split»
4. Скачиваешь готовые треки

## Форматы экспорта

| Формат | Описание |
|---|---|
| WAV 16-bit | Стандартный lossless, совместим везде |
| WAV 24-bit | Высокое разрешение для мастеринга |
| MP3 320 | Сжатый, удобен для дистрибуции |

## Управление

| Действие | Способ |
|---|---|
| Добавить маркер кликом | Включить режим «✦ Add Markers», кликнуть по волне |
| Добавить маркер на playhead | Клавиша `M` |
| Удалить маркер | Правая кнопка мыши по маркеру |
| Плей / Пауза | Клавиша `Space` |
| Переименовать трек | Кликнуть по названию в списке треков |

## Особенности

- Всё обрабатывается локально — файл не покидает браузер
- Сессия сохраняется: файл и маркеры восстанавливаются при перезагрузке страницы
- Если исходник не в 44.1 кГц — автоматически ресемплируется перед экспортом в WAV
- Каждый трек можно скачать отдельно или все сразу

## Технологии

- [React](https://react.dev/) + [Vite](https://vite.dev/)
- [wavesurfer.js](https://wavesurfer.js.org/) — визуализация и маркеры
- [@breezystack/lamejs](https://github.com/breezystack/lamejs) — MP3-энкодер на чистом JS
- Web Audio API / OfflineAudioContext — декодирование и ресемплинг
